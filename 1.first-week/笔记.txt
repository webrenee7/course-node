创建博客的步骤：
    1>.安装生成博客的工具
        npm install hexo-cli -g

    2>.生成博客
        e:
        mkdir xyBlogTest
        cd xyBlogTest
        hexo --help  显示帮助信息
        hexo init  下载博客

    3>.启动博客服务
        hexo server
        换端口：server -p 2222
    4>.停止博客服务
         ctrl c

发布到github
    1>.安装插件：（安装一次在当前项目下即可）
        npm install hexo-deployer-git --save
    2>.配置：
        deploy:
          type: git
          repo: https://账号:密码@github.com/webrenee7/webrenee7.github.io.git
          branch: master
          message: push
    3>.发布：hexo deploy
    4>.修改
        hexo g 重新生成（第一次不需要，以后每次发布前都需要重新生成代码）
    5>.重新发布
